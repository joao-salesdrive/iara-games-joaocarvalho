# iara-games-joaocarvalho

🎮 IARA Games
Contexto e proposta
A IARA Games é uma plataforma de distribuição de jogos digitais com compromisso exclusivo com desenvolvedoras brasileiras independentes. A proposta é centrada em impulsionar talentos emergentes, estabelecendo um espaço dedicado a produções locais — excluindo qualquer jogo vinculado a empresas internacionais.

Decisões de design, UX e acessibilidade
Paleta de cores
As cores foram escolhidas para criar contraste elevado e identidade forte. O par texto branco #FFFFFF sobre fundo escuro #242424 atingiu avaliação AAA (contraste 15.5), válido para todos os tamanhos e pesos de fonte — garantindo leitura confortável em qualquer cenário.
Tipografia
Foram escolhidas Geist para textos gerais e Geist Mono para títulos e destaques. A fonte mono reforça o caráter técnico e independente da plataforma, criando uma identidade visual coerente com o universo gamer.
Layout
O banner principal usa uma composição assimétrica (60/40) para dar destaque ao jogo principal sem esconder os demais. As seções de jogos utilizam cards quadrados com aspect-ratio: 1/1, garantindo consistência visual independente do tamanho da imagem original.
Botão de login
Posicionado com destaque no header usando a cor Azul Iara #3EA8DA, diferenciando-o visualmente dos links de navegação e facilitando o acesso rápido.
